# プライバシーポリシー

最終更新日: 2026-04-17

## 1. 対象サービス

本ポリシーは、Chrome 拡張機能 **「出社スケジュールマネージャー」** に適用されます。

この拡張機能は、組織内で運用される Google スプレッドシートの出社表を、Google Apps Script を経由して閲覧・更新し、未設定リマインドおよび司会通知を表示するためのものです。

## 2. 取得・利用する情報

本拡張機能は、以下の情報を取り扱います。

- ユーザーが設定した氏名
- 出社予定・勤務ステータス
- 名簿情報
  氏名、チーム等
- 司会予定情報
- 拡張機能の設定情報
  テーマ設定、表示設定、通知用キャッシュ等

## 3. 情報の取得方法

本拡張機能は、以下の方法で情報を取得します。

- ユーザーが拡張機能の初回設定画面・設定画面に入力した情報
- 組織内の Google Apps Script Web アプリへの通信結果
- 組織内の Google スプレッドシートから Google Apps Script を経由して返される情報

## 4. 情報の利用目的

取得した情報は、以下の目的にのみ利用します。

- 出社表の閲覧
- 出社表の更新
- 来週未設定日の検出と通知
- 朝会・夕会の司会予定の表示と通知
- チームメンバーの出社状況表示
- 表示高速化のためのローカルキャッシュ
- テーマや表示設定の保存

## 5. 情報の保存場所

- ユーザー端末内の `chrome.storage.local`
- 組織内の Google Apps Script / Google スプレッドシート

本拡張機能は、ブラウザの閲覧履歴、検索履歴、訪問ページ本文等を取得しません。

## 6. 第三者提供

本拡張機能は、取得した情報を広告、マーケティング、データ販売の目的で第三者に提供しません。

情報は、以下の場合に限り取り扱われます。

- 本拡張機能の機能提供のため、Google Apps Script および Google スプレッドシートを利用する場合
- 法令に基づき開示が必要な場合
- セキュリティ対応や不正利用防止のため必要な場合

## 7. 外部送信

本拡張機能は、以下のホストに対して通信を行います。

- `https://script.google.com/*`
- `https://script.googleusercontent.com/*`

これらは、組織内で運用される Google Apps Script Web アプリとの通信のためにのみ利用されます。

## 8. ユーザーによる管理

ユーザーは、Chrome の拡張機能設定から本拡張機能を削除できます。削除後、ブラウザ内に保存されたローカル設定およびキャッシュは Chrome の仕様に従って削除されます。

サーバー側のデータ修正・削除は、組織内の Google スプレッドシート運用ルールに従って行ってください。

## 9. セキュリティ

本拡張機能は、必要最小限の権限のみを使用し、Google Apps Script との通信には HTTPS を使用します。

## 10. Google API データの利用

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
